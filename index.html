Capstone Project: Friendship Plus

Project Summary

Friendship Plus is a web and mobile application designed to help students form meaningful connections in post-secondary institutions. The platform addresses challenges such as social anxiety, loneliness, and the struggle to find like-minded peers. It offers user-friendly features like profile creation, secure messaging, and an intelligent recommendation system to connect students based on shared interests.

Project Vision

The goal of Friendship Plus is to create a platform where post-secondary students can connect with others who share similar interests, fostering friendships and improving student well-being. By reducing loneliness and encouraging collaboration, the app enhances both mental health and academic productivity.

Project/Business Requirements

  • User Registration & Profile Management: Allow students to create accounts and personalize profiles.
  • Secure Messaging System: Implement private messaging for user communication.
  • Privacy Controls: Enable users to manage visibility and communication preferences.
  • Matching & Recommendation System: Suggest connections based on shared interests.
  • User Engagement Insights: Integrate analytics for app improvement.

Project Plan

The Friendship Plus project is structured to follow an iterative Agile development process with clearly defined milestones and tasks. The development cycle includes:

  • Planned Start Date: September 12, 2024
  • Planned End Date: March 28, 2025
  • Development Sprints: 10 sprints covering UI/UX, backend integration, testing, and final deployment.
  • Key Milestones: System design completion (October 10, 2024), Frontend & Backend integration (November 12, 2024), User acceptance testing (November 16, 2024).

Requirements Analysis and Design

The Friendship Plus application is designed to provide post-secondary students with a social networking platform tailored to academic needs. The system follows strict privacy guidelines, ensuring that user information remains secure and only accessible upon mutual consent.

  • User Registration: Secure 2FA authentication using university email.
  • Profile Customization: Users can update their bio, upload profile pictures, and list their interests.
  • Search & Connect: Users can find and connect with others based on shared interests.
  • Event Notifications: Upcoming university events are displayed to encourage participation.
  • Privacy & Security: Personal information is hidden until both users accept a connection request.
